НЕОБХОДИМАЯ ВЕРСИЯ ИНТЕРПРЕТАТОРА: python 3.9.0

При первой установке необходимо создать виртуальную среду и установить зависимости:   
        python -m venv env
        .\env\Scripts\activate
        python -m pip install -U pip
        pip install -r requirements.txt
        Это установит все необходимые зависимости.
	Убедитесь, что используется компилятор виртуальной среды.

Для запуска сервера:
streamlit run .\Home.py

helpers: модули, необходимые для работы модели
output: подготовленная модель и shap-значения
pages: логика страниц аналитической платформы
Home.py: точка входа в приложение
requirements.txt: необходимые зависимости для корректной работы решения
