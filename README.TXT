При первой установке необходимо:
    Если имеется готовый env -> активировать его
    Иначе:
        py -m venv env
        env\Scripts\activate
        py -m pip install -U pip
        pip install -r requirements.txt

        Это установит все необходимые зависимости.
        Рекомендуемый компилятор: python 3.11.4

АНАЛИЗ ДАННЫХ находится в data_research.ipynb
ИНФЕРЕНС находится в inference.py
КОД ОБУЧЕНИЯ используемой модели находится в boosting_models/lightgbm.ipynb
Приложение Flask находится в папке server

boosting_models: модели градиентного бустинга
classic_models: классические алгоритмы
data: датасеты
helpers: наши модули, необходимые для работы с моделью
hyperparameteres: подбор гиперпараметров
output: содержит дамп лучшей модели и её предсказания
submissions: папка для сохранения предсказаний моделей
visualization: папка с гистограммами и матрицами рассеяния
requirements.txt: необходимые зависимости для корректной работы решения

server:
    server.py -> загружает инференс модели и использует его для предсказаний
                 класса для признаков, передаваемых в виде POST-запроса.
    template: шаблоны html
    static: статические файлы
